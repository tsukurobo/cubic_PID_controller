# Readme

このライブラリは、Cubic用のPID制御ライブラリです。

バージョン0.14は、バージョン1.7のCubicに依存しています。
このバージョンはアルファ版です。

- ver 0.13

- ver 0.12

  - サンプルプログラムでput, sendしていなかったのを修正
